# 🏡 House Price Prediction ML App

This is a Machine Learning project that predicts house prices using XGBoost and Streamlit.

## 📊 Dataset
Kaggle House Prices Dataset (Ames Housing)

## 🧠 Model Used
- XGBoost Regressor
- StandardScaler
- Feature Engineering + One-Hot Encoding

## 🚀 Features
- User-friendly Streamlit UI
- Real-time prediction
- High accuracy ML model (~90% R² score)
